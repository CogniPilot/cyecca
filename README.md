# cyecca

**Cy**mbolic **E**stimation and **C**ontrol with **C**omputer **A**lgebra

A Python library for simulation and analysis of Modelica models, with support for Lie group structures (SE(2,3), SO(3), etc.).

## Architecture

```
Modelica (.mo) → Rumoca (BLT, causalization) → DAE/Explicit IR JSON → Cyecca → Simulation/Analysis
```

**Cyecca** is a simulation/analysis tool. **Rumoca** handles compilation (parsing, flattening, BLT, causalization).

## Quick Start

```python
from cyecca.io import load_dae_ir_json
from cyecca.backends.casadi import CasadiBackend

# Load model from DAE IR JSON (generated by Rumoca)
with open("model.json") as f:
    model = load_dae_ir_json(f.read())

# Simulate
backend = CasadiBackend(model)
t, sol = backend.simulate(t_final=10.0, dt=0.01)

# Linearize
A, B, C, D = backend.linearize()
```

## Backends

| Backend | Use Case |
|---------|----------|
| **CasADi** | Fast numerics, C code generation, optimization |
| **SymPy** | Symbolic analysis, LaTeX export, analytical derivatives |
